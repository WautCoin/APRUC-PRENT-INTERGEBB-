# APRUC-PRENT-INTERGEBB-
API

## Getting started

```bash
npm install
npm start
```

The server starts on `http://localhost:3000` by default. Set the `PORT` environment variable to use a different port.

## Endpoints

| Method | Path      | Description          |
|--------|-----------|----------------------|
| GET    | /health   | Health-check endpoint |
